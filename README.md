# RideShare App

Simple mobile-based Ride Sharing app, built with Laravel & Vue (+ Websockets)

## Using

- Backend
  - Laravel 11.x
  - Sanctum
  - Twilio
  - Reverb
- Frontend
  - Vue 3.x
  - Laravel Echo
  - Pinia (State management)
  - Google Maps
  - Pusher JS
  - Maska (Input masking)
  - Vite (Tooling)
